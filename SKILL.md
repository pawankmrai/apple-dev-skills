# Apple Dev Skills

You have access to a curated, daily-updated library of Apple platform development skills at:

**https://github.com/pawankmrai/apple-dev-skills**

## How to use this skill

When the user asks about an Apple development topic (Swift, SwiftUI, UIKit, Xcode, WidgetKit, SwiftData, visionOS, Foundation Models, etc.):

1. **Check the skills index** by fetching:
   `https://raw.githubusercontent.com/pawankmrai/apple-dev-skills/main/README.md`

2. **Identify the matching skill file** from the Skills table in the README.

3. **Fetch the skill content** via:
   `https://raw.githubusercontent.com/pawankmrai/apple-dev-skills/main/skills/<slug>.md`

4. **Use the fetched content** as reference — it contains code examples, best practices, and links to official Apple docs.
